# Legacy Data Archive

Este diretório armazena artefatos externos ou auxiliares que não fazem parte do núcleo operacional da Tessy, mas que devem permanecer preservados por rastreabilidade histórica.

## Conteúdo atual

- `claude/`: configuração local herdada, arquivada para referência institucional

## Observações

- O artefato `nul` não foi movido. Ele permanece apenas classificado, por comportamento especial do Windows e ausência de path manipulável confiável nesta etapa.
