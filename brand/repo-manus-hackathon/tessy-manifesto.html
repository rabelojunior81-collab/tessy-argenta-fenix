O Produto

Tudo que você precisa.
Nada que você não precisa.

Tessy não é um chat com IA colado num editor. É um ambiente de engenharia completo: filesystem local, terminal PTY real, GitHub integrado, vault criptográfico — e um LLM que entende o contexto de cada um deles.

GitHub Integrado Nativo
Listar, buscar, commitar, criar branches e PRs — diretamente pela IA, com aprovação do usuário para ações destrutivas. 10 tools nativas.
✅ Entregue
LLM com Function Call Loop
Pipeline de IA com intent detection + iteração de tools (até 10x). Gemini Pro como motor principal. A IA age, não só responde.
✅ Entregue
LLM Gateway Multi-Provider
Google Cloud, OpenRouter, Anthropic, local (Ollama) — um endpoint inteligente que roteia pelo modelo certo para cada tarefa.
🔧 Em Construção
Filesystem Local (File System Access API)
CRUD completo de arquivos no seu disco. Sem upload, sem cloud. Construído em React 19 + TypeScript. A IA lê, edita e cria — você aprova.
✅ Entregue
Terminal PTY Real
Não é sandbox. É um shell real via node-pty + WebSocket, com sessão autenticada e spawn no diretório do projeto. TypeScript end-to-end.
✅ Entregue
Vault Criptográfico (AES-GCM 256)
Suas API keys criptografadas no IndexedDB com PBKDF2. Dual-mode: transparente ou protegido por senha. Build com Vite para máxima performance.
✅ Entregue

GitLab AI Hackathon 2026

Tessy como
Agente Autônomo GitLab.

Para o hackathon, Tessy age como um agente autônomo acionado por eventos do GitLab via webhooks. Três capacidades demonstradas com a plataforma GitLab Duo Agent.

Análise de Merge Request
O agente clona a branch, analisa as mudanças, roda testes com Vitest e comenta no MR com um relatório de qualidade gerado pelo Gemini Pro.
Auto-Documentação
Em cada push para a main, o agente gera ou atualiza a documentação do projeto com base no diff do código. Zero esforço manual.
Análise de Segurança
Usando Gemini Pro no Google Cloud, o agente escaneia o código por vulnerabilidades conhecidas e abre uma issue no GitLab se encontrar algo crítico.
