# Skills Ativas

## Disponíveis neste ambiente
- **keybindings-help** — customizar atalhos de teclado do Claude Code
- **simplify** — revisar código alterado para qualidade, reuso e eficiência
- **claude-api** — construir apps com Claude API / Anthropic SDK

## Ferramentas de Agente
- **Explore** — exploração de codebase (quick / medium / very thorough)
- **Plan** — planejamento arquitetural de implementações
- **general-purpose** — pesquisa e tarefas multi-step
- **claude-code-guide** — dúvidas sobre Claude Code, Agent SDK, API

## Notas de Uso
- Preferir ferramentas dedicadas (Read, Grep, Glob) antes de Bash
- Paralelizar tool calls quando não há dependência entre elas
- Confirmar antes de ações destrutivas ou que afetam estado compartilhado
