## 3. Como Funciona a Onisciência da Tessy

Pipeline completo (já funcionava antes, só estava bloqueado pelo bug de File System):

```
Usuário clica "Vincular Pasta"
  → showDirectoryPicker() → FileSystemDirectoryHandle
  → WorkspaceContext.selectDirectory()
  → directoryHandle armazenado no estado

Usuário conversa com Tessy CoPilot
  → ChatContext passa directoryHandle para gemini/service.ts
  → Se directoryHandle disponível → WORKSPACE_FUNCTION_DECLARATIONS ativas
  → Tessy pode chamar:
      workspace_read_file(file_path)          → lê conteúdo de arquivo
      workspace_list_directory(directory_path) → lista entradas de pasta
      workspace_search_files(query, extension) → busca texto no workspace
  → Mudanças (create/edit/delete) → aprovação explícita do usuário
```
