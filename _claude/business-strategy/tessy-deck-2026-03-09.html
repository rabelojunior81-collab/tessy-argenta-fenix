O Produto

Tudo que você precisa.
Nada que você não precisa.

Tessy não é um chat com IA colado num editor. É um ambiente de engenharia completo: filesystem local, terminal PTY real, GitHub integrado, vault criptográfico — e um LLM que entende o contexto de cada um deles.

🖥️
Terminal PTY Real
Não é sandbox. É um shell real via node-pty + WebSocket, com sessão autenticada e spawn no diretório do projeto.
✅ Entregue
📁
Filesystem Local (File System Access API)
CRUD completo de arquivos no seu disco. Sem upload, sem cloud. A IA lê, edita e cria — você aprova.
✅ Entregue
🐙
GitHub Integrado Nativo (10 tools)
Listar, buscar, commitar, criar branches e PRs — diretamente pela IA, com aprovação do usuário para ações destrutivas.
✅ Entregue
🔐
Vault Criptográfico (AES-GCM 256)
Suas API keys criptografadas no IndexedDB com PBKDF2. Dual-mode: transparente ou protegido por senha.
✅ Entregue
🤖
LLM com Function Call Loop
Pipeline de IA com intent detection + iteração de tools (até 10x). A IA age, não só responde.
✅ Entregue
🔀
LLM Gateway Multi-Provider
OpenRouter, Anthropic, Minimax, local (Ollama) — um endpoint inteligente que roteia pelo modelo certo para cada tarefa.

FAQ

Perguntas frequentes

Atualmente Tessy é um projeto proprietário do Rabelus Lab. O código não é público, mas o protocolo de desenvolvimento (TDP/TSP/TMP) e a metodologia são abertos. Há discussão ativa sobre abrir partes do core como open source no futuro — especialmente o LLM Gateway.
Tessy roda inteiramente no browser via React + Vite. Seus projetos, conversas e configurações ficam no IndexedDB do browser (no seu dispositivo). API keys são criptografadas com AES-GCM 256 bits + PBKDF2. O único dado que sai do seu ambiente é o texto enviado ao LLM provider que você escolher — o que é inevitável para qualquer ferramenta de IA.
Cursor é um editor de código (extensão / app local) centrado no ato de escrever código. Claude Code é uma CLI — funciona no terminal, sem interface visual. Tessy é browser-native, visual, e centrado no projeto como um todo — filesystem, git, GitHub, terminal e IA em uma tela, sem instalar nada. Para o dev que quer controle real sem a complexidade de CLI, Tessy ocupa um espaço que nenhum dos dois cobre.
Hoje, sim — Gemini é o provider padrão e você precisa de uma API key do Google AI Studio (gratuita para uso básico). Com o lançamento do multi-provider LLM, você poderá usar OpenRouter (com acesso a 200+ modelos), Anthropic, ou modelos locais via Ollama — sem depender do Google.
No estágio atual, buscamos principalmente co-fundadores e colaboradores bootstrap — não investimento dilutivo.
