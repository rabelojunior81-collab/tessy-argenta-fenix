## PREMISSA ESTRATÉGICA

Tessy v4.9.1 "Tesseract" possui base técnica de nível A+. O próximo movimento não é técnico — é de negócio. A janela de oportunidade no mercado de Vibe Coding está aberta agora. O objetivo desta série de documentos é mapear como capturar essa janela com os recursos disponíveis hoje: produto funcional, fundador técnico, zero budget de marketing, e uma legião de agentes IA para execução.

**Norte estratégico único:**
> Tessy é o ambiente de desenvolvimento IA mais completo e local-first para o developer brasileiro semi-profissional — com gateway de LLMs, terminal real e metodologia própria.

## SÍNTESE DO MAPA DE OPORTUNIDADES

### O que Tessy já tem (ativos de negócio, não só técnicos)

| Ativo | Valor de mercado |
|-------|-----------------|
| Terminal real no browser | Unicórnio técnico — nenhum Lovable/Bolt entrega |
| Vault criptográfico local | Diferencial de privacidade em mercado cloud |
| Metodologia TDP/TSP/TMP | Não copiável — é método, não código |
| 10 GitHub tools nativas | Workflow completo sem sair do ambiente |
| Filesystem real via FSA API | Controle que SaaS genérico não oferece |
| Identidade LiquidGlass | Visual distinto, reconhecível |
| Português nativo | Vantagem competitiva no mercado BR |

### As 5 Apostas Estratégicas (ordem de prioridade)

1. **LLM Gateway / Multi-provider** — Transformar o lock-in Gemini em vantagem: ser o hub de todos os LLMs
2. **Mercado Brasileiro** — Posicionamento "Claude Code brasileiro" com preço em reais
3. **Early Adopter Community** — Primeiros 100 usuários viram evangelistas e co-criadores
4. **Affiliate Stack** — Receita passiva sem esforço técnico, a partir da semana 1
5. **Hackathon → Time** — Construir co-founders e contribuidores via bootstrap + equity

### Horizonte de Tempo

```
Semana 1-4:    Affiliate links ativos + Early Adopter Program lançado
Mês 1-3:       Primeiros 100 usuários, conteúdo recorrente, 1 hackathon
Mês 3-6:       Tessy Indie (plano pago em BRL), LLM multi-provider aberto
Mês 6-12:      Time formado, 500+ usuários, LLM Gateway como feature-flag
Ano 2:         Tessy como plataforma/infraestrutura para outros builders
```

---

## PRINCÍPIOS QUE GUIAM ESTA ESTRATÉGIA

1. **Bootstrap-first**: Crescimento sem depender de investimento externo inicial
2. **Community-led**: Os primeiros usuários são parceiros, não consumidores
3. **Revenue-from-day-1**: Affiliate stack gera receita antes de ter produto "completo"
4. **BR-market-first**: Dominar o nicho antes de ir global
5. **Content > Ads**: Posicionamento orgânico é mais sustentável que tráfego pago
6. **Equity over salary**: Time construído por participação futura, não custo fixo
