# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus on the work, not the person.
- Share context generously.

## Unacceptable behavior

- Harassment, hate speech, or intimidation.
- Deliberate sabotage of the repository structure.
- Repeated attempts to overwrite independent repositories or ignore local project rules.

## Enforcement

Maintainers may remove content or restrict access when behavior undermines the project or its collaborators.

## Scope

This code of conduct applies to all repository interactions, issues, pull requests, and discussions tied to this master repo.
