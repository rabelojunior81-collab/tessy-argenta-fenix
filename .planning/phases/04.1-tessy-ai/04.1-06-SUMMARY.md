## Implementation Details

### StreamingCursor Component
Animated blinking cursor (530ms rate) using purple accent color (#7C5CFF). Hidden when not streaming, visible during streaming for visual feedback.

### CancelStreamButton Component
Red button with X icon that calls `cancelStream()` to abort the in-flight streaming request via AbortController.

### ToolCallCard Component
Displays tool name badge, formatted JSON args in code block, and approve/reject buttons. Shows pending/approved/rejected status states.

### ChatContext Streaming Pipeline
- Added imports for `ProviderRegistry`, `ToolBroker`, `builtInTools`, `ChatMessage`
- Added `abortControllerRef` and `brokerRef` for streaming lifecycle
- Enhanced `cancelStream()` to abort in-flight requests and clear state
- Added `streamWithProvider()` function that:
  - Creates `AbortController` for cancellation
  - Initializes `ToolBroker` with built-in tools
  - Streams text chunks in real-time, updating UI immediately
  - Accumulates tool call deltas and shows pending tool calls
  - Executes approved tools via `broker.invoke()`
  - Re-injects tool results into conversation and continues stream
  - Non-Gemini providers use streaming; Gemini preserves existing flow

### CoPilot Streaming UI
- Imports `StreamingCursor`, `CancelStreamButton`, `ToolCallCard`
- Shows `CancelStreamButton` instead of send button when streaming
- During streaming, displays markdown-rendered text with animated cursor
- Renders `ToolCallCard` components for each pending tool call

### Server Endpoints
- `POST /ai/proxy/ollama` - Ollama CORS proxy with streaming NDJSON pipe
- `POST /ai/tools/execute` - Server-side tool execution via PTY for runCommand
