## Decisions Made

- Used `FileSystemDirectoryHandle.resolve()` instead of string comparison for path containment (research §6.2 pitfall #4)
- Used Zod `schema.safeParse()` for validation (research §9 — "Don't Hand-Roll")
- Secret hydration traverses JSON replacing `${KEY}` patterns via `getSecret(key)` resolver
- ToolCallDispatcher handles both delta-style (OpenAI/Anthropic) and complete-style (Gemini/Ollama)
- runCommand DENYLIST includes: `rm -rf`, `git push --force`, `DROP DATABASE`, `curl|wget pipe-to-sh`, fork bomb, `sudo su`, `chmod 777`
