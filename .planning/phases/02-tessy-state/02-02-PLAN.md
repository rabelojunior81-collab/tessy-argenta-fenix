<threat_model>
## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| User filesystem -> FileExplorer | Handles e nomes de arquivo renderizados na UI | file entries |
| Session storage -> expanded tree | Paths relativos persistidos reabrem arvore | expanded paths |
| Keyboard/input -> file actions | Enter/Space/context menu disparam acoes | row commands |

## Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-02-A | Information Disclosure | expandedPaths persistence | mitigate | Persistir apenas paths relativos ao workspace atual, nunca path absoluto do sistema. |
| T-02-02-B | Tampering | keyboard row actions | mitigate | Distinguir arquivo/pasta e manter delete sempre atras de confirmacao explicita. |
| T-02-02-C | Denial of Service | large file open | mitigate | Manter `fileOpenPolicy` e modal antes de `setSelectedFile`. |
| T-02-02-D | Accessibility Regression | FileExplorer rows | mitigate | Usar roles/labels/aria-selected/aria-expanded testados por RTL. |
</threat_model>
