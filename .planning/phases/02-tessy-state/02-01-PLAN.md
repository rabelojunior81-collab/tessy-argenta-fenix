## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| Browser storage -> React state | Dados persistidos em sessoes anteriores influenciam estado atual | session envelope |
| File System Access handle -> selected file | Permissao/handle controla restauracao de arquivo | relative file path and content read |
| Terminal transcript -> storage | Saida visual do terminal vira historico limitado | terminal lines |

## Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01-A | Information Disclosure | sessionPersistence | mitigate | Sanitizar e bloquear `content`, `token`, `secret`, `absolutePath`, `stdout` e transcript acima de 200 linhas. |
| T-02-01-B | Tampering | LayoutContext restore | mitigate | Validar viewer contra allowlist e descartar payload invalido. |
| T-02-01-C | Spoofing/DoS | Workspace restore | mitigate | Revalidar handle/permissao antes de restaurar arquivo e degradar para estado `missing`. |
| T-02-01-D | Elevation of Privilege | RealTerminal restore | mitigate | Restaurar transcript visual sem conectar WebSocket/PTY automaticamente. |
</threat_model>
