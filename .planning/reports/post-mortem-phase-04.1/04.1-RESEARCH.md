### 2.1 OpenAI / OpenRouter (SSE)

**Endpoint:** `POST /v1/chat/completions`
**Headers:**
```
Content-Type: application/json
Authorization: Bearer <API_KEY>
```

**Request Body:**
```json
{
  "model": "gpt-4o",
  "messages": [
    { "role": "system", "content": "<system prompt>" },
    { "role": "user", "content": "<user message>" }
  ],
  "stream": true,
  "tools": [...],
  "max_tokens": 4096,
  "temperature": 0.2
}
```

**Response SSE (`text/event-stream`):**
```
data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":1234567890,"model":"gpt-4o","choices":[{"index":0,"delta":{"role":"assistant","content":""},"finish_reason":null}]}

data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":1234567890,"model":"gpt-4o","choices":[{"index":0,"delta":{"content":"Olá"},"finish_reason":null}]}

data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":1234567890,"model":"gpt-4o","choices":[{"index":0,"delta":{"content":""},"finish_reason":"stop"}]}

data: [DONE]
```

**Tool-call delta (OpenAI):**
```
data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":1234567890,"model":"gpt-4o","choices":[{"index":0,"delta":{"tool_calls":[{"index":0,"id":"call_xxx","type":"function","function":{"name":"readFile","arguments":"{\"path\":\""},"finish_reason":null}]},"finish_reason":null}]}

data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":1234567890,"model":"gpt-4o","choices":[{"index":0,"delta":{"tool_calls":[{"index":0,"id":"call_xxx","function":{"arguments":"\"./src/index.ts\"}"}],"finish_reason":null}]}
```

**Parsing Strategy:**
```typescript
// eventsource-parser stream — NÃO EventSource API
const eventStream = response.body
  .pipeThrough(new TextDecoderStream())
  .pipeThrough(createParser(stream))
  .getReader();

for await (const event of eventStream) {
  if (event.data === '[DONE]') break;
  const parsed = JSON.parse(event.data);
  const delta = parsed.choices?.[0]?.delta;
  if (delta?.content) yield { kind: 'text', text: delta.content };
  if (delta?.tool_calls) {
    for (const tc of delta.tool_calls) {
      yield { kind: 'tool_call_delta', toolCallId: tc.id, toolName: tc.function.name, argDelta: tc.function.arguments };
    }
  }
}
```

**Endpoint:** `POST /v1/messages`
**Headers:**
```

```

**Request Body:**
```json
{
  "model": "claude-sonnet-4-6",

  ],
  "stream": true,
  "max_tokens": 4096,

```

**Response SSE (`text/event-stream`):**
```
event: message_start
data: {"type":"message_start","message":{"id":"msg_xxx","type":"message","role":"assistant","content":[],"stop_reason":null}}

event: content_block_start
data: {"type":"content_block_start","index":0,"content_block":{"type":"text","text":""}}

event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"text_delta","text":"Olá"}}

event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"input_json_delta","partial_json":"{\"path\":\""}}

event: message_stop
data: {"type":"message_stop","message":{"id":"msg_xxx","type":"message","role":"assistant","content":[{"type":"tool_use","id":"toolu_xxx","name":"readFile","input":{"path":"./src/index.ts"}}],"stop_reason":"end_turn"}}
```

**Tool-call delta (Anthropic):**
```
event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"input_json_delta","partial_json":"{\"path\":\"./src/app"}}
```

**Parsing Strategy:**
```typescript
const stream = response.body
  .pipeThrough(new TextDecoderStream())
  .pipeThrough(createParser(stream));

const partialArgs = new Map<string, string>();

for await (const event of stream) {
  if (event.event === 'content_block_delta') {
    const d = JSON.parse(event.data);
    if (d.delta.type === 'text_delta') {
      yield { kind: 'text', text: d.delta.text };
    }
    if (d.delta.type === 'input_json_delta') {
      const id = String(d.index);
      partialArgs.set(id, (partialArgs.get(id) ?? '') + d.delta.partial_json);
      yield { kind: 'tool_call_delta', toolCallId: id, argDelta: d.delta.partial_json };
    }
  }
  if (event.event === 'message_stop') yield { kind: 'done' };
}
```

### 2.3 Gemini (SSE com `?alt=sse`)

**Endpoint:** `POST /v1beta/models/<model>:generateContent?alt=sse`
**Headers:**
```
Content-Type: application/json
Authorization: Bearer <API_KEY>
```

**Request Body:**
```json
{
  "contents": [{ "parts": [{ "text": "user message" }] }],
  "systemInstruction": { "parts": [{ "text": "system prompt" }] },
  "tools": [{ "functionDeclarations": [...] }],
  "generationConfig": {
    "temperature": 0.2,
    "maxOutputTokens": 4096,
    "responseModalities": ["TEXT"]
  },
  "stream": true
}
```

**Response SSE (`text/event-stream`):**
```
data: {"promptFeedback":{"blockReason":0},"candidates":[{"contentCreationTime":"1234567890","refusalLevel":0,"avgLogProbs":0}]}

data: {"candidates":[{"content":{"parts":[{"text":"Olá"}]},"finishReason":"0","index":0}]}

data: {"candidates":[{"content":{"parts":[{"functionCall":{"name":"readFile","args":{"path":"./src"}}}]},"finishReason":"1","index":0}]}
```

**Parsing Strategy:**
```typescript
// Gemini MANDATORY: ?alt=sse query param
// Gemini chunks: texto vem como "text" ou "functionCall" (obj completo, não delta)
const stream = response.body.pipeThrough(new TextDecoderStream()).pipeThrough(createParser(stream));

for await (const event of stream) {
  const parsed = JSON.parse(event.data);
  const candidate = parsed.candidates?.[0];
  if (!candidate) continue;
  const part = candidate.content?.parts?.[0];

  if (part?.text) yield { kind: 'text', text: part.text };
  if (part?.functionCall) {
    // Gemini envia objeto COMPLETO por chunk — NÃO é delta
    yield {
      kind: 'tool_call_complete',
      toolCallId: String(part.functionCall.name), // Gemini não tem ID — usa name
      toolName: part.functionCall.name,
      args: part.functionCall.args
    };
  }
}
```

### 2.4 Ollama (NDJSON — NÃO SSE)

**Endpoint:** `POST /api/chat`
**Headers:**
```
Content-Type: application/json
```

**Request Body:**
```json
{
  "model": "qwen2.5-coder:14b",
  "messages": [{ "role": "user", "content": "hello" }],
  "stream": true,
  "tools": [...]
}
```

**Response (NDJSON — newline-delimited JSON, NÃO SSE):**
```
{"model":"qwen2.5-coder:14b","created_at":"2026-04-30T00:00:00Z","message":{"role":"assistant","content":"Olá"},"done":false}
{"model":"qwen2.5-coder:14b","created_at":"2026-04-30T00:00:00Z","message":{"role":"assistant","content":""},"done":true}
```

**Tool-call (Ollama):**
```
{"model":"qwen2.5-coder:14b","created_at":"...","message":{"role":"assistant","content":""},"tool_calls":[{"type":"function","function":{"name":"readFile","arguments":{"path":"./src/index.ts"}}}],"done":false}
{"model":"qwen2.5-coder:14b","created_at":"...","done":true}
```

**Parsing Strategy:**
```typescript
// Ollama: NDJSON, não SSE —usa TextDecoderStream + readLine
const decoder = new TextDecoderStream();
const reader = response.body.pipeThrough(decoder).getReader();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  // Ler linha completa (Ollama termina com \n)
  const lines = value.split('\n').filter(Boolean);
  for (const line of lines) {
    if (!line.trim()) continue;
    const parsed = JSON.parse(line);
    const msg = parsed.message;

    if (msg?.content) yield { kind: 'text', text: msg.content };
    if (msg?.tool_calls) {
      for (const tc of msg.tool_calls) {
        yield {
          kind: 'tool_call_complete',
          toolCallId: tc.function.name, // Ollama não tem ID
          toolName: tc.function.name,
          args: tc.function.arguments
        };
      }
    }
    if (parsed.done) yield { kind: 'done' };
  }
}
```

## 6. Tool System — Detalhamento

### 6.1 ToolSpec e Zod Schema

```typescript
// src/lib/ai/tools/ToolSpec.ts
import { z } from 'zod';

export interface ToolSpec<TArgs = unknown, TResult = unknown> {
  name: string;
  description: string;
  schema: z.ZodType<TArgs>;
  invoke(
    args: TArgs,
    ctx: {
      sandbox: WorkspaceSandbox;
      ui: ToolUI;
      signal: AbortSignal;
    }
  ): Promise<ToolResult<TResult>>;
}

export interface ToolResult<T = unknown> {
  isError: boolean;
  content: string;
  data?: T;
}

// ToolUI: interface para UI durante execução
export interface ToolUI {
  requestDiffApproval(ctx: {
    path: string;
    before: string;
    after: string;
  }): Promise<boolean>;
  showProgress?(message: string): void;
}
```

```typescript
import { z } from 'zod';
import type { ToolSpec, ToolResult } from '../ToolSpec';

export const ReadFileArgs = z.object({
  path: z.string().min(1).refine(p => !p.includes('\0'), 'null-byte proibido'),
```

#### writeFile

```typescript
// src/lib/ai/tools/builtins/writeFile.ts
import { z } from 'zod';
import type { ToolSpec, ToolResult } from '../ToolSpec';

export const WriteFileArgs = z.object({
  path: z.string().min(1).refine(p => !p.includes('\0'), 'null-byte proibido'),
  content: z.string(),
  encoding: z.enum(['utf-8', 'utf-16']).default('utf-8'),
});

export const writeFileTool: ToolSpec<z.infer<typeof WriteFileArgs>> = {
  name: 'writeFile',
  description: 'Escreve ou sobrescreve um arquivo no workspace. Requer aprovação do usuário via diff preview.',
  schema: WriteFileArgs,
  async invoke(args, { sandbox, ui, signal }) {
    await sandbox.assertInside(args.path);

    // LÊ ANTES para gerar diff — não escreve antes da aprovação
    const before = await sandbox.readFileText(args.path).catch(() => '');

    // requestDiffApproval é UM MÉTODO do ToolUI — não é blocking call
    // O ToolCallDispatcher pausa o stream e aguarda resolução do usuário
    const approved = await ui.requestDiffApproval({
      path: args.path,
      before,
      after: args.content,
    });

    if (!approved) {
      return { isError: true, content: 'Operação cancelada pelo usuário.' };
    }

    await sandbox.writeFileText(args.path, args.content, args.encoding);
    return {
      isError: false,
      content: `Arquivo escrito: ${args.path} (${args.content.length} caracteres)`,
      data: { path: args.path, chars: args.content.length },
    };
  },
};
```

#### runCommand

```typescript
// src/lib/ai/tools/builtins/runCommand.ts
import { z } from 'zod';
import type { ToolSpec, ToolResult } from '../ToolSpec';

const DENYLIST = [
  /^rm\s+-rf\b/,
  /^git\s+push\s+--force$/,
  /^git\s+push\s+-f$/,
  /^:\(\)\{:\|:&\};/,
  /^DROP\s+DATABASE/i,
  /^DROP\s+TABLE/i,
  /^curl\s+.+\|\s*sh$/,
  /^wget\s+.+\|\s*sh$/,
];

export const RunCommandArgs = z.object({
  command: z.string().min(1),
  cwd: z.string().optional(), // se omitido, usa workspace root
});

export const runCommandTool: ToolSpec<z.infer<typeof RunCommandArgs>> = {
  name: 'runCommand',
  description: 'Executa um comando shell no workspace. Comandos destrutivos exigem confirmação.',
  schema: RunCommandArgs,
  async invoke(args, { sandbox, ui, signal }) {
    const cwd = args.cwd ?? sandbox.workspaceRoot;

    // Denylist check
    const isDenied = DENYLIST.some(r => r.test(args.command));
    if (isDenied) {
      const confirmed = await ui.requestDestructiveConfirmation({
        action: 'runCommand',
        command: args.command,
        reason: 'Comando potencialmente destrutivo detectado.',
      });
      if (!confirmed) {
        return { isError: true, content: 'Comando bloqueado pela denylist.' };
      }
    }

    // Executa via broker local (server/index.ts)
    const response = await fetch('http://127.0.0.1:3002/ai/tools/run-command', {
      method: 'POST',
      signal,
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        sessionToken: sandbox.sessionToken,
        command: args.command,
        cwd,
      }),
    });

    if (!response.ok) {
      const error = await response.text();
      return { isError: true, content: `Erro ao executar comando: ${error}` };
    }

    const result = await response.json();
    return {
      isError: false,
      content: result.output ?? result.error ?? 'Comando executado.',
      data: { exitCode: result.exitCode, signal: result.signal },
    };
```

```typescript
// src/lib/ai/tools/WorkspaceSandbox.ts
export class WorkspaceSandbox {
  constructor(
    private handle: FileSystemDirectoryHandle,
    private sessionToken: string // para delegar ao broker quando necessário
  ) {}

  async assertInside(childPath: string): Promise<void> {
    // 1. Null-byte check
    if (childPath.includes('\0')) {
      throw new ToolError('E_PATH', 'Caminho contém byte nulo.');
    }

    // 2. Normaliza e tenta resolver
    const normalized = path.normalize(childPath);

    // 3. Usa FSA resolve() — único anchor seguro
    // Se child está fora do handle, resolve() retorna null ou array vazio
    const parts = await this.handle.resolve(normalized);
    if (!parts || parts.length === 0) {
      throw new ToolError('E_PATH', `Caminho "${childPath}" está fora do workspace.`);
    }

    // 4. Verifica que o resolved path ainda está dentro do handle
    // ( FSA resolve não撒谎 se houver symlinks, mas normalização ajuda)
    const resolved = path.join(this.workspaceRoot, ...parts);
    const rootResolved = path.join(this.workspaceRoot, normalized);
    if (!resolved.startsWith(this.workspaceRoot)) {
      throw new ToolError('E_PATH', 'Tentativa de escape de sandbox via symlink ou path especial.');
    }
  }

  async readFileText(path: string, encoding: BufferEncoding = 'utf-8'): Promise<string> {
    await this.assertInside(path);
    const handle = await this.handle.getFileHandle(path.split('/').pop());
    const file = await handle.getFile();
    return new TextDecoder(encoding).decode(await file.arrayBuffer());
  }

  async writeFileText(path: string, content: string, encoding: BufferEncoding = 'utf-8'): Promise<void> {
    await this.assertInside(path);
    const fileName = path.split('/').pop();
    const handle = await this.handle.createWritable();
    await handle.write(new TextEncoder().encode(content));
    await handle.close();
  }

  async enumerateDir(path: string): Promise<string[]> {
    await this.assertInside(path);
    const dirHandle = await this.handle.getDirectoryHandle(path, { create: false });
    const entries: string[] = [];
    for await (const [name] of dirHandle.entries()) {
      entries.push(name);
    }
    return entries;
  }

  get workspaceRoot(): string {
    // Retorna o path absoluto do handle (se disponível via FileSystemDirectoryHandle)
    return 'MEMORY_PATH'; // Preencher com path real do workspace
  }
}
```

### 6.4 ToolBroker + Secret Hydration

```typescript
// src/lib/ai/tools/ToolBroker.ts
import type { ToolSpec } from './ToolSpec';

interface SecretStore {
  get(provider: string, key: string): Promise<string | null>;
}

export class ToolBroker {
  private registry = new Map<string, ToolSpec>();
  private secrets: SecretStore;

  constructor(secrets: SecretStore) {
    this.secrets = secrets;
    // Registra builtins
    this.register(readFileTool);
    this.register(writeFileTool);
    this.register(listDirectoryTool);
    this.register(runCommandTool);
    this.register(searchFilesTool);
    this.register(readGitFileTool);
  }

  register<T extends ToolSpec>(tool: T): void {
    this.registry.set(tool.name, tool);
  }

  async invoke(
    toolName: string,
    rawArgs: unknown,
    ctx: { sandbox: WorkspaceSandbox; ui: ToolUI; signal: AbortSignal }
  ): Promise<ToolResult> {
    const tool = this.registry.get(toolName);
    if (!tool) {
      return { isError: true, content: `Ferramenta "${toolName}" não encontrada.` };
    }

    // 1. Zod validation (2 retries)
    const MAX_RETRIES = 2;
    let parsed = tool.schema.safeParse(rawArgs);
    let retries = 0;

    while (!parsed.success && retries < MAX_RETRIES) {
      const errorMsg = parsed.error.issues
        .map(i => `${i.path.join('.')}: ${i.message}`)
        .join('; ');
      // Envia erro para o modelo tentar corrigir
      // (pattern: volta para o stream como mais um turn)
      retries++;
      if (retries >= MAX_RETRIES) {
        return { isError: true, content: `E_TOOL_ARGS: ${errorMsg}` };
      }
      // Pattern: retorna erro estruturado e aguarda retry do modelo
      return { isError: true, content: `E_TOOL_ARGS_RETRY: ${errorMsg}` };
    }

    // 2. Hydrate secrets
    const hydratedArgs = this.hydrateSecrets(parsed.data);

    // 3. Execute
    try {
      return await tool.invoke(hydratedArgs, ctx);
    } catch (err) {
      if (err instanceof ToolError) {
        return { isError: true, content: err.message };
      }
      return { isError: true, content: `E_INTERNAL: ${(err as Error).message}` };
    }
  }

  private hydrateSecrets(args: unknown): unknown {
    // Substitui placeholders como ${GITHUB_TOKEN} por valores do IndexedDB
    // Padrão: detecta strings com padrão ${PROVIDER.KEY}
    return JSON.parse(
      JSON.stringify(args).replace(
        /\$\{(GITHUB|ANTHROPIC|OPENAI|GEMINI)_(TOKEN|KEY)\}/g,
        (_, provider, key) => {
          // Busca no secret store (IndexedDB via Dexie)
          return this.secrets.get(provider.toLowerCase(), key.toLowerCase()) ?? '';
        }
      )
    );
  }
}
```

### 6.5 ToolCallDispatcher

```typescript
// src/lib/ai/tools/ToolCallDispatcher.ts
import type { AsyncIterable } from 'AsyncIterable';

export interface DispatchResult {
  text: string;
  toolCalls: Array<{
    toolName: string;
    toolCallId: string;
    args: unknown;
    status: 'pending' | 'approved' | 'rejected';
  }>;
}

export class ToolCallDispatcher {
  private toolCalls = new Map<string, ToolCallState>();

  async consume(
    stream: AsyncIterable<ChatChunk>,
    broker: ToolBroker,
    ctx: ToolContext
  ): Promise<DispatchResult> {
    let text = '';
    const toolCallArgs = new Map<string, string>();

    for await (const chunk of stream) {
      switch (chunk.kind) {
        case 'text':
          text += chunk.text;
          break;
        case 'tool_call_delta':
          // Concatena fragmento JSON (OpenAI/Anthropic)
          const existing = toolCallArgs.get(chunk.toolCallId!) ?? '';
          toolCallArgs.set(chunk.toolCallId!, existing + chunk.argDelta);
          break;
        case 'tool_call_complete':
          // Gemini/Ollama: objeto completo
          toolCallArgs.set(chunk.toolCallId!, JSON.stringify(chunk.args));
          break;
        case 'done':
          break;
      }
    }

    // Parse e valida todos tool-calls
    const resolvedToolCalls: DispatchResult['toolCalls'] = [];
    for (const [toolCallId, argsJson] of toolCallArgs.entries()) {
      const [toolName, ...rest] = toolCallId.split('_');
      try {
        const args = JSON.parse(argsJson);
        resolvedToolCalls.push({
          toolName,
          toolCallId,
          args,
          status: 'pending',
        });
      } catch {
        // JSON parcial — inválido
        resolvedToolCalls.push({
          toolName,
          toolCallId,
          args: null,
          status: 'rejected',
        });
      }
    }

    return { text, toolCalls: resolvedToolCalls };
  }
}
```

```typescript
// src/lib/ai/context/TokenCounter.ts
import { encodingForModel } from 'js-tiktoken';

const MODEL_LIMITS: Record<string, number> = {
  'gpt-4o': 128_000,
  'gpt-4o-mini': 128_000,
  'gpt-5': 200_000,
  'claude-sonnet-4-6': 200_000,
  'claude-haiku-4-5-20251001': 200_000,
  'gemini-3-pro': 1_000_000,
  'gemini-3-flash': 1_000_000,
  'qwen2.5-coder:14b': 128_000, // Ollama — aproxima
};

export class TokenCounter {
  private encoders = new Map<string, ReturnType<typeof encodingForModel>>();

  async count(messages: ChatMessage[], model: string): Promise<number> {
    // OpenAI: usa tiktoken
    if (model.startsWith('gpt-')) {
      const encoder = await this.getEncoder(model);
      let total = 0;
      for (const msg of messages) {
        total += encoder.encode(msg.content).length;
        total += 4; // per-message overhead
      }
      total += 3; // assistant overhead
      return total;
    }

    // Anthropic: usa endpoint próprio (se disponível) ou heurística
    if (model.startsWith('claude-')) {
      // Heurística: ~4 chars por token
      return messages.reduce((sum, m) => sum + m.content.length / 4, 0);
    }

    // Gemini/Ollama: heurística simples
    return messages.reduce((sum, m) => sum + m.content.length / 4, 0);
  }

  async estimate(messages: ChatMessage[], model: string): Promise<{ used: number; limit: number }> {
    const used = await this.count(messages, model);
    const limit = MODEL_LIMITS[model] ?? 100_000;
    return { used, limit };
  }

  private async getEncoder(model: string) {
    if (!this.encoders.has(model)) {
      this.encoders.set(model, encodingForModel('o200k_base'));
    }
    return this.encoders.get(model)!;
  }
}
```

```typescript
// src/lib/ai/context/SlidingWindow.ts
export class SlidingWindow {
  private systemPrompt: ChatMessage | null = null;

  async compact(
    messages: ChatMessage[],
    model: string,
    counter: TokenCounter,
    onCompact: (reason: string) => void
  ): Promise<ChatMessage[]> {
    const { used, limit } = await counter.estimate(messages, model);

    if (used < limit * 0.8) {
      return messages; // Dentro do budget — não mexe
    }

    // 1. System prompt: SEMPRE preservado
    const systemMsg = messages.find(m => m.role === 'system');
    if (systemMsg) {
      this.systemPrompt = systemMsg;
    }

    // 2. Últimas N mensagens: preservadas
    const RECENT_COUNT = 8;
    const recent = messages.slice(-RECENT_COUNT);

    // 3. Mensagens intermediárias: substituídas por resumo
    const middle = messages.slice(0, -RECENT_COUNT);
    if (middle.length > 0 && this.systemPrompt) {
      const summary = await this.summarize(middle);
      const summaryMsg: ChatMessage = {
        role: 'system',
        content: `[Compactado] Resumo da conversa anterior:\n${summary}`,
      };

      onCompact(`Compactadas ${middle.length} mensagens.`);

      return [this.systemPrompt, summaryMsg, ...recent];
    }

    return messages;
  }

  private async summarize(messages: ChatMessage[]): Promise<string> {
    // Usa modelo barato para resumir (Haiku ou gpt-5-mini)
    // Implementação: chama ProviderRegistry com modelo de compactação
    // Por ora: retorna placeholder string
    return messages.length > 0
      ? `[${messages.length} mensagens anteriores foram compactadas]`
      : '';
  }
}
```

## 9. Don't Hand-Roll

| Problema | Não Recomendar | Usar |
|---------|----------------|------|
| SSE parsing | `EventSource` API nativa | `eventsource-parser/stream` |
| Tool schema | JSON Schema manual | `zod` + `zod-to-json-schema` |
| Token counting | Estimativas manuais | `js-tiktoken` (OpenAI) + endpoint Anthropic |
| Provider abstraction | SDKs vendor (Vercel AI SDK, LangChain) | Fetch pura + ProviderStyle interface |
| Streaming | Bibliotecas que acumulam buffer | Web Streams nativa (`ReadableStream`) |
| Secret storage | Variáveis de ambiente no client | IndexedDB via Dexie (RISCO_ACEITO documentado) |

## 10. Common Pitfalls

| # | Pitfall | Symptom | Fix |
|---|---------|---------|-----|
| 1 | Esquecer `?alt=sse` no Gemini | Provider retorna JSON array, não SSE | Adicionar query param sempre |
| 2 | Concatenar `tool_call.arguments` para Gemini/Ollama | JSON corrompido — eles enviam objeto completo | Detectar provider e tratar diferente |
| 3 | Usar `EventSource` API | Não suporta POST nem custom headers | `fetch` + `ReadableStream` + `eventsource-parser` |
| 4 | Validar path com `startsWith` | Falha com Unicode, `..`, null-byte | `FileSystemDirectoryHandle.resolve()` |
| 5 | Não tratar terminator Ollama (`done:true`) | Stream nunca termina | NDJSON parsing com `done` flag check |
| 6 | Não passar `AbortSignal` | Usuário cancela mas request continua | Passar `AbortController.signal` para todo fetch |
| 7 | Tokenizer único | Token count errado para modelos não-OpenAI | `TokenCounter` router per-provider |
| 8 | Truncamento silencioso do system prompt | Modelo perde regras de sandbox | SlidingWindow preserva system sempre |
